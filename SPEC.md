# 时差计算器 - 规格说明书

## 1. 项目概述

- **项目名称**: 时差计算器 (Time Difference Calculator)
- **项目类型**: 单页Web应用
- **核心功能**: 输入一个时区及其时间，自动计算并显示另一个时区对应的时间
- **目标用户**: 需要经常处理不同时区时间转换的用户

## 2. UI/UX 规格

### 布局结构

- 单页面垂直居中布局
- 主体卡片式容器，包含两个时区输入区域
- 整体宽度最大 480px，自适应移动端

### 视觉设计

**色彩方案**
- 背景: #0f0f1a (深邃夜空色)
- 卡片背景: #1a1a2e (深紫蓝)
- 主色调: #00d4ff (科技青)
- 次要色: #7c3aed (活力紫)
- 文字主色: #e0e0e0
- 文字次色: #888888
- 边框高亮: rgba(0, 212, 255, 0.3)

**字体**
- 标题: "Orbitron", sans-serif (科技感)
- 正文: "Noto Sans SC", "Microsoft YaHei", sans-serif

**间距**
- 卡片内边距: 32px
- 元素间距: 20px
- 输入框内边距: 12px 16px

**视觉效果**
- 卡片: 圆角 16px，微妙阴影
- 输入框: 圆角 8px，聚焦时边框发光效果
- 按钮: 渐变背景，圆角 8px
- 全局: 柔和的辉光效果

### 组件

**时区输入卡片 (x2)**
- 标签: "时区A" / "时区B"
- 时区下拉选择器 (基于Intl API)
- 日期输入框 (type="date")
- 时间输入框 (type="time")

**结果展示区**
- 显示时区B的计算结果时间
- 清晰的大字体显示
- 显示时差信息

**操作按钮**
- "计算" 按钮 - 主按钮样式
- "交换时区" 按钮 - 次要按钮样式

## 3. 功能规格

### 核心功能

1. **时区选择**
   - 提供所有常用时区选项 (基于浏览器Intl API)
   - 默认时区A: 北京时间 (Asia/Shanghai)
   - 默认时区B: 纽约时间 (America/New_York)

2. **时间输入**
   - 日期选择器，支持选择任意日期
   - 时间选择器，支持选择任意时间（精确到分钟）
   - 默认显示当前日期时间

3. **时间转换计算**
   - 选择时区A的日期和时间后
   - 选择时区B的时区
   - 自动计算并显示时区B对应的本地时间

4. **交换功能**
   - 一键交换时区A和时区B的位置
   - 便于反向计算

### 用户交互流程

1. 用户进入页面，默认显示当前时间
2. 用户选择/修改时区A
3. 用户选择时区A的日期和时间
4. 用户选择时区B（目标时区）
5. 系统自动计算并显示结果

### 边界情况处理

- 跨日期边界（如北京早上8点对应纽约前一天晚上7点）
- 夏令时自动处理
- 空值时显示友好提示
